## 功能特点

1. **端点配置灵活**：输出端点和输入端点可以选择不同的配置，端点类型和最大包长可以不同。
2. **支持多种传输类型**：端点类型支持中断传输（interrupt）和块传输（bulk）。

4. **界面优化**：对界面进行了优化，提升了用户体验。
